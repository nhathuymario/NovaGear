# 🚀 NovaGear Frontend - Library Stack

## 📦 Core & Build Tools (Dev)
- **Vite**: Build tool cực nhanh.
- **TypeScript**: Ép kiểu dữ liệu (đã đổi từ .jsx sang .tsx).
- **Tailwind CSS**: Framework CSS utility-first để code giao diện nhanh.
- **PostCSS/Autoprefixer**: Hỗ trợ Tailwind tương thích mọi trình duyệt.

> **Lệnh cài:** > `npm install -D typescript @types/react @types/react-dom tailwindcss postcss autoprefixer`

---

## 🛠 Main Libraries (Dependencies)
- **React Router Dom**: Quản lý điều hướng (trang chủ, login, profile...).
- **Axios**: Gọi API đến Backend Spring Boot.
- **Lucide React**: Bộ icon SVG dạng component rất nhẹ.
- **Ant Design (antd)**: Bộ UI Component (Button, Table, Modal...) có sẵn.
- **Zustand**: Quản lý State toàn cục (thay cho Redux vì nhẹ hơn).
- **React Hook Form**: Xử lý validate Form (Login/Register) cực mượt.

> **Lệnh cài:** > `npm install react-router-dom axios lucide-react antd zustand react-hook-form`
